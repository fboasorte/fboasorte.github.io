Tag <col>

Definição e Uso

A tag <col> especifica as propriedades da coluna para cada coluna dentro de um elemento <colgroup>.

A tag <col> é útil para aplicar estilos a colunas inteiras, em vez de repetir os estilos para cada célula, para cada linha.

Exemplo

Define a cor de fundo das três colunas com as tags <colgroup> e <col>: 

              
                  <table>
                      <colgroup>
                          <col span="2" style="background-color:red">
                          <col style="background-color:yellow">
                      </colgroup>
                      <tr>
                          <th>ISBN</th>
                          <th>Título</th>
                          <th>Priço</th>
                      </tr>
                      <tr>
                          <td>3476896</td>
                          <td>Meu primeiro HTML</td>
                          <td>R$53</td>
                      </tr>
                </table>

Atributos

Atributo Valor Descrição
span number Especifica o número de colunas que um elemento <col> deve abranger
