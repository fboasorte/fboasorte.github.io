Definição e Uso

A tag <img> é usada para incorporar uma imagem em uma página HTML.

As imagens não são tecnicamente inseridas em uma página da web; as imagens estão vinculadas a páginas da web. A tag <img> cria um espaço de retenção para a imagem referenciada.

A tag <img> tem dois atributos obrigatórios:

src - especifica o caminho para a imagem alt - especifica um texto alternativo para a imagem, se a imagem por algum motivo não puder ser exibida Nota: Além disso, sempre especifique a largura e a altura de uma imagem. Se a largura e a altura não forem especificadas, a página pode piscar enquanto a imagem é carregada.

Dica: Para vincular uma imagem a outro documento, simplesmente aninhe a tag <img> dentro de uma tag <a> (veja o exemplo abaixo).



Exemplo

Como inserir uma imagem:

            
              <img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">

Exemplo

Como adicionar um hiperlink a uma imagem:

            
              <a href="https://www.w3schools.com">
                <img src="w3html.gif" alt="W3Schools.com" width="100" height="132">
              </a>
