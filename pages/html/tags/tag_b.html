Definição e Uso

A tag <b> define um texto em negrito sem dar-lhe importancia extra.

Exemplo

Uma imagem embutida: 
              
                 <p>Isso é um texto normal - <b>e esse é um texto negrito</b>.</p>

Atributos

A tag <b> suporta os atributos globais.

A tag <b> suporta os atributos de eventos.
