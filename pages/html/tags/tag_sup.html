Definição e Uso

A tag <sub> define um texto superescrito. Texto superescrito aparece meio caractere acima da linha normal, e é algumas vezes rederizado em fonte menor. Texto superescrito pode ser usado para algumas notas, como WWW[1]
Dica: Para usar texto subescrito use a tag <sub>

Exemplo

Uma imagem embutida: 
              
                <p>Press <kbd>Ctrl</kbd> + <kbd>C</kbd> to copy text (Windows).</p>

                <p>Press <kbd>Cmd</kbd> + <kbd>C</kbd> to copy text (Mac OS).</p>
