Tag <base>

Referencias Home Sobre

Definição e Uso

A tag <base> especifica o URL base e / ou destino para todos os URLs relativos em um documento.

A tag <base> deve ter um href ou um atributo target presente, ou ambos.

Só pode haver um único elemento <base> em um documento e deve estar dentro do elemento <head>.

Exemplo

Especifique um URL padrão e um destino padrão para todos os links em uma página: 

                
                    <head>
                        <base href="https://www.w3schools.com/" target="_blank">
                    </head>
                    
                    <body>
                        <img src="images/stickman.gif" width="24" height="39" alt="Stickman">
                        <a href="tags/tag_base.asp">HTML Tag base</a>
                    </body>

Atributos

Atributo Valor Descrição
href URL Especifica o URL base para todos os URLs relativos na página
target _blank
_parent
_self
_top		 Especifica o destino padrão para todos os hiperlinks e formulários na página
