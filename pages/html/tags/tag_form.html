Definição e Uso

A tag <form> é usada para criar um formulário HTML para o usuário preencher. Uma tag <form> pode conter um ou mais dos seguintes elementos:

  • <input>
  • <textarea>
  • <button>
  • <select>
  • <option>
  • <select>
  • <option>
  • <optgroup>
  • <fieldset>
  • <label>
  • <output>

Exemplo

Uma imagem embutida: 
              
                <form action="/action_page.php" method="get">
                    <label for="fname">First name:</label>
                    <input type="text" id="fname" name="fname"><br><br>
                    <label for="lname">Last name:</label>
                    <input type="text" id="lname" name="lname"><br><br>
                    <input type="submit" value="Submit">
                </form>

Atributos

Atributo Valor Descrição
accept-charset character_set Especifica a codificação de caracteres usadas na submissão do formulário.
action URL Especifica para onde enviar os dados quando o formulário é submetido.
autocomplete on
off		 Especifica se o formulário deveria ter autocompletar automaticamento ou não.
enctype application/x-www-form-urlencoded
multipart/form-data
text/plain		 Especifica como os dados dados submetidos deveriam ser codigicados para o servidor(apenas para o método POST)
method get
post 		Especifica o método HTTP usado para enviar os dados.
name text Especifica o nome do formulário.
novalidate novalidate Especifica que o formulário nao deveria ser validados quando submetido.
rel external
help
license
next
nofollow
noopener
noreferrer
opener
prev
search		 Especifica a relação entre o recurso ligado e o documento atual.
target _blank
_self
_parent
_top		 Especifica onde exibir a resposta que é recebida depois de submeter o formulário.
