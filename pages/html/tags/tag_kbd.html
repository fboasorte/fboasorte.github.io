Definição e Uso

A tag <kbd> é usada para definir entrada do teclado. O conteúdoi é mostrado com a fonta monospace padrão do navegador.
Dica: Essa tag não está depreciada. No entanto, é possível conseguir efeitos mais rico usando CSS.

Exemplo

Uma imagem embutida: 
              
                <p>Press <kbd>Ctrl</kbd> + <kbd>C</kbd> to copy text (Windows).</p>

                <p>Press <kbd>Cmd</kbd> + <kbd>C</kbd> to copy text (Mac OS).</p>

Atributos
