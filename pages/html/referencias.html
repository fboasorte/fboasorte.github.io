Tag Descrição
<base> Especifica a URL base/alvo para todos os URLs relativos em um documento.
<col> Especifica as propriedades da coluna para cada coluna em um elemento <colgroup>.
<colgroup> Especifica um grupo de uma ou mais colunas em uma tabela para formatação.
<script> Define um script no lado do cliente.
<th> Define uma célula de cabeçalho em uma tabela.
<ul> Define uma lista não ordenada.
<embed> Define um espaço para uma aplicação externa.
<sup> Define um texto superescrito.
<kbd> Define entrada do teclado.
<fieldset> Agrupa elementos relacionados dentro de um formulario
<form> Define um formulário HTML para o usuário preencher
<b> Define um texto em negrito sem dar-lhe importancia extra.
<noscript> Define um conteúdo alternativo a ser exibido para usuários que desativaram/não usportam scripts em seus navegadores.
<ol> Define uma lista ordenada, que pode ser em ordem numérica ou alfabética.
<picture> Oferece aos desenvolvedores da web mais flexibilidade na especificação de recursos de imagem.
<iframe> Exibi uma página da web dentro de uma página da web.
<optgroup> É usada para agrupar opções relacionadas.
<img> Vai incorporar uma imagem em uma página HTML.
